# DVW Resort Explorer

Aplicacion web estatica (HTML, CSS y JavaScript) para explorar resorts de DreamVacationWeek.

## Estructura esperada del repositorio

```text
.
  index.html
  assets/
    css/styles.css
    js/app.js
  data/
    resorts.json
    continents.json
  README.md
```

## Probar en local

No abras con `file://`.

Usa un servidor local en esta carpeta:

```bash
python -m http.server 8000
```

Abre:

```text
http://localhost:8000/
```
